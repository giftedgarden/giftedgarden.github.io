## Brand Rules Used

- Primary purple: `#7A3E91`
- Elephant purple: `#B166C8`
- Light lavender: `#D7B2E5`
- Garden green: `#C9E7A6`
- Sunshine yellow: `#F7D65C`
- Warm white: `#FFFDFB`
- Charcoal: `#343434`
- Lavender sprigs are the repeating botanical language.
- One golden four-point sparkle is the signature detail.
- The tagline remains outside the primary logo.
- Real child photography should only be posted when the appropriate permissions
  are on file.

## Generation Notes

The artwork was created with OpenAI's built-in image generation using the
official Gifted Garden horizontal logo and elephant mascot as visual references.
The prompt set required a warm, polished child-care identity, readable rounded
lettering, the approved purple/green/yellow palette, lavender-only botanical
motifs, one golden sparkle, generous safe margins, and no hearts, dots, extra
stars, micro flowers, or watermarks. The provider post additionally used the
approved Dionne Panton portrait with instructions to preserve her identity.
